# Istruzioni
Se stai alla prima volta che compili allora:
0. Runnare report ed eliminare tutto tranne il tex
1. vai nella cartella include -> Formatter
2. Titlepage.tex e premi play
3. andare su Report-frn.tex e premere play
5. infine runnare il Report.tex

Se invece non hai mai cambiato niente all'interno di titlepage.tex puoi runnare solo report

Nel caso invece in cui modifichi titlepage va rifatta tutta la scaletta eliminando i file contenenti Report-frn nel nome
1. Eliminare Report.pdf
2. Eliminare tutto ciò che ha a nel nome Report-frn
3. vai nella cartella include -> Formatter
4. Titlepage.tex e premi play
5. andare su Report-frn.tex e premere play
6. infine runnare il Report.tex
